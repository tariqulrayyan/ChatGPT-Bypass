## Requirements

<b>Export API Key </b>

Head over to 'beta.openai.com' to retrieve your personal API key and set it the variable CHATGPT_TOKEN...or hard code the key. idc

Windows:

````
$env:CHATGPT_TOKEN='<api key>'
````

Nix/MAC:

````
export CHATGPT_TOKEN=<api key>
````
<b>Install jq </b>

Mac:

````
brew install jq
````

Debian:

````
apt install jq
````
